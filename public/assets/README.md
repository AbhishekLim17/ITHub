# Assets Folder

This folder contains all static assets for the certified IT website.

## Required Images

Add the following images to this folder:

### Favicons
- `favicon.ico` - Main favicon (32x32)
- `favicon-16x16.png` - Small favicon
- `favicon-32x32.png` - Medium favicon
- `apple-touch-icon.png` - iOS home screen icon (180x180)

### Open Graph
- `og-image.jpg` - Social media preview image (1200x630)

### Content Images
- `office.jpg` - Office/team photo
- `logo.png` - Company logo (transparent background)
- `hero-bg.jpg` - Hero section background (optional)

### Partner Logos
- `cisco-logo.png`
- `microsoft-logo.png`
- `iso-logo.png`
- `vmware-logo.png`

## Image Optimization Tips

1. Use WebP format for modern browsers
2. Keep file sizes under 200KB
3. Use appropriate dimensions (no oversized images)
4. Include alt text for all images
5. Use lazy loading for below-the-fold images
